Common Git Commands

A minimal, accessible example page with table, details, and a little JavaScript for filtering.

Command Description Example
git init Initialize a repository in the current folder. git init
git clone Copy a remote repository to your machine. git clone https://example.com/repo.git
git add Stage changes for the next commit. git add .
git commit -m Create a snapshot of staged changes. git commit -m "feat: add login"
git status Show modified/staged files and current branch. git status
git log View commit history (use --oneline --graph --all for a compact graph). git log --oneline --graph --all
git diff Show line-by-line changes before committing. git diff
git branch List/create/delete branches. git branch feature/login
git switch Switch to another branch. git switch feature/login
git merge Combine changes from another branch. git merge feature/login
git push Upload local commits to the remote. git push origin main
git pull Fetch and merge remote changes. git pull origin main
Tips
  • Use git log --oneline to quickly list commits.
  • Create descriptive branch names, e.g., feature/payment-api.
  • Write clear commit messages (imperative mood, short subject, optional body).
